# 🧠 Memory Game

### Installation
📦️ node_modules
```sh
npm install
```

### Preview
🌐 http://localhost:3000/
```sh
npm run dev
```

### Build
📦️ src ➡️ 🎁 dist
```sh
npm run build
```

### Deploy
☁️ Set your remote or config [gh-pages](https://www.npmjs.com/package/gh-pages)
```sh
npm run deploy
```
